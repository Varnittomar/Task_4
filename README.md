# Task 4

## Description

This project demonstrates a simple webpage created using HTML and CSS. The page contains an image displayed inside a div container and a styled button with a gradient background.

## Files Included

* index.html
* style.css
* README.md

## Features

* Proper HTML document structure
* External CSS styling
* Image display inside a container
* Gradient styled button
* Equal left and right spacing (25%)
* Top and bottom spacing (10%)
* Box-sizing property implemented
* No Flexbox used
* No relative or absolute positioning used
* No overflow:hidden used

## How to Run

1. Download or clone the project.
2. Keep all files in the same folder.
3. Open index.html in any web browser.

## Technologies Used

* HTML5
* CSS3
